04 / Architecture

Frontend structure

Semantic tokens power every component surface, so theme changes require variable overrides instead of component rewrites.

Playground data lives in script.js, while component contracts live in reusable classes inside styles.css.

The page is static, deployable from GitHub Pages, and has no runtime dependency beyond the browser.
